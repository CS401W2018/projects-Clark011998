1. Why do we need a viewport meta tag?

According to what I heard earlier in class, it is for controlling how our webapge look in the mobile device. To make sure it is responsive in diffferent screen sizes.

2. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

"flex-direction: row" will take effect on the html, or it will take priority because it comes later in the css code.

3. How do you change the CSS for a web page displayed on a phone?

We can change the CSS for a web page displayed on a phone by using media queries inside the CSS. Just like what we did in the class earlier when we made the code "@media only screen and (max-width: 650px) {}".
