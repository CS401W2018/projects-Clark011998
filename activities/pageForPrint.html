Engagement - 10/23/24 - Page for Print

1. Why is the viewport meta tag important?

It is important because it controls how the webpage is displayed on different devices, especially mobile devices, like what we have discussed earlier in the class. And in this way, it would be accessible to everyone in everywhere or any devices they use.

2. How can you adjust the CSS for printing?

By creating another CSS file for printing options. You can type the function "display:none" to the part where you don't want it to include while printing

3. When would you want to adjust the CSS for printing?

When your know that your webpage is going to be used to print by the user. And also when we want to remove unnecessary elements.
