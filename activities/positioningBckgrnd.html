1. What is the difference between a block and an inline tag? Give examples of each one.

Block tags are like big boxes that take up the width of the page, and they always start on a new line and push other content down. Inline tags are small portions that are inside a line, such as words or even images. Sample of block tags are div tags, paragraph tags, and h1 tags. And for Inline tag samples are span tag, a tag, and also strong tags.

2. What is normal flow in HTML?.

Normal flow in HTML refers to the default way elements are placed in the code. Like the ones we use to do before in class. It also defines how block and inline tags are placed in the code.

3. What does float do? Why do you need to clear the float?

Float is usually used inside the CSS file. It is used for positioning an element, it may be left or right. We need to clear the float so that it would not overlap over things.

4. What do absolute, fixed, and relative do?

These are positioning methods, it allows us to move things as we want to inside the webpage. Absolute is placed in the nearest element or the page itself. Fixed is when we want the specific element to stay in the same place even when users scrolls through the webpage. Relative is positioned to where it normally would be, space is preserved in the layout.
