Engagement - 11/06/2024 - Form

1. The form tag has an "action" attribute. What does it do?

It takes you to another site where the details that you put will be viewed. It takes all the information in another site.

2. The form tag has a "method" attribute. What does it do?

Inside the "method" attribute is we can either use post or get as the functions. It is also specifies how the form data will be sent to te server when the user wants to submit it.

3. What is the difference between a "POST" and a "GET"?

We can use POST for submitting data securely. GET is for retrieving and querying data. Reminder, that if we use POST, it does not mean that is 100% secured, it does not act as the security.
