1. What you learned about Flex and CSS.

Flex is used to making webpage layouts inside CSS. It helps arrange the items that are placed inside the webpage. It kind of look the same with Grid but it is way better to create layouts that can look good on different screen sizes.

2. Why do you need to have an accessible website? What does that mean?

To help the user make the webpage more easy to access. It could help them solve the problem that they have been searching for. Also for the reputation of the owner of the website itself.
