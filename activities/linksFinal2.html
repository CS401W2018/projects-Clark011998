Engagement - Links 09.20.2024

Explain the difference between an absolute link and a relative link. Why would you use relative links?

Absolute link - it could be other websites or even our own websites
Relative link - is your/our own website only.
We use relative links if we want to use our own work only. This will let the user see the work we have been doing.

Do links always need to be inside a NAV tag?

No, links does not always have to be inside a NAV tag

List the 3 rules with relative links.
  1. Periods (.) - means current folder
  2. Two Periods (..) - parent folder
  3. Slash (/) - meaning at the beginning, start at the root
