Internet and Web definitions

Internet

Internet or sometimes can be referred to as net, is a global network that connects millions of devices across the world. Through the internet, it also made communication across the globe possible. Websites, email, social medias, online gaming, and other applications are examples of how we can make use of the internet.

Web

World wide web or Web in short is a system of interconnected documents and resources that are accessed through the internet. Web is also a part of the internet that is composed of various other websites and it can be accessed through web browsers.

How are the Internet and Web different?

The Internet and Web are relate but they differ in concepts. The internet is a larger global network that supports different kinds of services including the Web. On the other hand, Web is a very specific kind of service inside the internet that allows people to use and interact with different websites through the browsers.

ASCII and Unicode

ASCII (American Standard Code for Information Interchange) and Unicode are both used in encoding. ASCII is an old way of using the code and it has a limit, while Unicode is the modern more advanced system, and has no limit.

What is UTF-8

UTF-8 (Unicode Transformation Format - 8 bit) is a flexible way and is a universal encoding that does not use a huge amount of storage space.

URL

URL (Uniform Resource Locator) is a web address of a specific location of a file or source on the web. The example for this is what we do when we do cloning in GitHub, we copy the URL and paste it in VScode to have the same copy of codes.

Domain Name

Domain Name is the address of the website we see in the browser. It is what the users type in the address bar of the browser, like for example www.facebook.com, the domain name is facebook.com

HTTP

HTTP (HyperText Transfer Protocol) is the foundation of how data is delivered across the web. When the users visit a website, it is the first thing that we see in the address bar.

HTTPS

HTTPS (HyperText Transfer Protocol Secure) is the secure version of HTTP, meaning it keeps our online data safe and private, especially on the sites that needs our peronal information.

IP Address

Internet Protocol Address is a unique number given to devices that are connected to the network or the internet. It acts as the users device's "address".

DNS

DNS (Domain Name System) makes the users browsing easy by converting easy to remember domain names into IP addresses.

W3C Recommendations

W3C (World Wide Web Consortium) is an international organization that keeps the web developed and maintain the standards, that it would always be accessible to everyone who uses it.

XML, XHTML, HTML, HTML5

Each of theses plays a role in the development of the web. XML is for data storage. XHTML is a very strict version of HTML. HTML is used for making webpages. HTML5 is the modern version of HTML with advanced features.
